Robotics & Embedded Systems Projects

Remote Controlled Car

Arduino based RC car programmed using C++. The robot is powered by 4x AA batteries, and uses an IR sensor module alongside a remote to be able to control movement of the 4x 5V DC motors.

Autonomous Line Follower Robot

Lead programmer and pseudocode designer for a group engineering project, where we were tasked to program an mbot to follow various unseen line paths.

Software Engineering & Game Design Projects

The Flames of Fusion

A text-based RPG made in python which showcases object oriented programming for enemies, abilities, and map locations. The game uses a grid-based modular world system with multiple locations, interactions and events throughout.

Sacrifice to Survive

A 2D unity game programmed using C#. The game is a survivors-like where you must make sacrifices throughout your playthrough to gain new powers and survive hordes of enemies.

Beach Environment Tech Demo

A simple 3D Unity game used to test rendering, terrain generation, and 3D Vector movement of both a player and third person camera objects.
